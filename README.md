# Löwen Play Pokerraum-Guide Deutschland

Static multi-page visitor guide for physical poker events. Locale: `de-DE`; country: Deutschland; currency: EUR; timezone: Europe/Berlin.

## Local preview

Run `python3 -m http.server 8080` in this folder and open `http://localhost:8080/`.

## Before publication

1. Replace every `{{VERIFIED_*}}` marker in `script.js`, HTML metadata and `sitemap.xml` with verified operator data.
2. Confirm the legal right to use the brand name and all local image assets.
3. Add only lawfully licensed event, venue and broadcast information.
4. Connect forms to a secure verified endpoint. Until then, forms do not claim submission.
5. Replace `noindex,nofollow` and `Disallow: /` only after legal, content and technical review.
6. Verify address, opening hours, age limits and applicable gambling permissions for the actual physical venue.

This project is presented as an independent guide and does not copy official brand artwork.
